## Resumo

O nosso projecto é sobre um sistema de gestão para uma pet shop.   
Faz a marcação, registo, e geração de documentos para os clientes.  
O acesso ao sistema será restringido só para os usuarios devidamente cadastrados no sistema.



## Extrutura da pagina

O projeto tem 2 pastas basicas:

- `src`: pasta do codigo
- `lib`: pasta das dependencias/livrarias

Meanwhile, the compiled output files will be generated in the `bin` folder by default.


## Classes

**Clientes** - todos os usuarios são criados atravez dessa classe. Será tambem a superclasse de vetenerario e user do sistema.  
**Vetenerario** - subclasse de clientes contem a informação dos vetenerarios do Pet Shop.   
**User** - São todos que tem acesso direto ao nosso sistema 
**Pets** - contem as informações do animal.  
**Historico** - contem o historico das consultas assim como toda a informação das consultas feitas.  
**Consulta** - informações das consultas.
